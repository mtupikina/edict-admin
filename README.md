# Edict Admin

Angular frontend for the Edict Admin application (electronic dictionary administration).

## Setup

```bash
npm install
```

Copy `src/environments/environment.ts` and ensure `apiUrl` points to your backend (default: `http://localhost:3000`).

## Development

```bash
npm start
```

Runs at `http://localhost:4200`.

## Build

```bash
npm run build
```

## Tests

```bash
# Unit tests
npm test

# E2E tests (run `npm start` in another terminal first)
npm run e2e
```
